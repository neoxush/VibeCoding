This folder contains a Tampermonkey user script designed to enhance Chrome's **Split Tabs View** feature (supported since Chrome version 119).  
The script seamlessly opens links in the designated split-view tab when users select the split mode, making it ideal for those who need to work with two pages simultaneously.
