## For Different Use Cases

### If You Want Standard Mode:
Just turn OFF "Road Mode" - all other defaults still work great!

### If You Want More Variety:
- Increase "Space Size Variation" to 0.8
- Click "Randomize Seed"

### If You Want Wider Street:
- Increase "Road Width" to 12m

### If You Want Sparser:
- Increase "Spacing" to 5m
