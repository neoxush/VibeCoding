# 2D to 3D Sketch Tool

A browser-based drawing application that allows users to create 2D sketches and convert them into 3D models with extrusion.

## Features

- **2D Drawing Tools**: Pen, Line, Rectangle, Circle
- **Real-time 3D Conversion**: Transform 2D sketches into extruded 3D models
- **Interactive 3D Preview**: Rotate and view your 3D models
- **Clean Interface**: Simple, intuitive design for easy use

## Getting Started

1. Open `index.html` in a modern web browser
2. Use the toolbar to select drawing tools
3. Draw your sketch on the left canvas
4. Click "Convert to 3D" to see your sketch as a 3D model
5. Drag to rotate the 3D model in the preview pane

## Tools

- **Pen**: Freehand drawing
- **Line**: Draw straight lines
- **Rectangle**: Draw rectangles
- **Circle**: Draw circles
- **Clear**: Clear the canvas
- **Convert to 3D**: Transform your 2D sketch into a 3D model

## Technical Stack

- HTML5 Canvas for 2D drawing
- Three.js for 3D rendering and extrusion
- Vanilla JavaScript for application logic
- CSS3 for styling

## Browser Support

Works in all modern browsers that support:
- HTML5 Canvas
- WebGL
- ES6 JavaScript features
